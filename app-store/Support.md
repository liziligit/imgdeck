# ImgDeck 技术支持

ImgDeck 是一款原生 macOS A4 图片排版工具，支持导入 1～9 张图片、选择版式、预览并保存为 PNG 或 JPG。

## 系统要求

- macOS 13.0 或更高版本；
- 当前发布目标以 App Store 页面显示的兼容机型为准。

## 基本使用方法

1. 点击“添加图片”，选择 1～9 张图片。
2. 使用“上移”“下移”调整图片顺序。
3. 选择需要的 A4 版式。
4. 设置分辨率与 DPI/DPCM 单位。
5. 点击“预览”检查效果。
6. 选择 PNG 或 JPG，点击“保存图片”。

## 常见问题

### 为什么有些位置是白色？

当图片数量少于所选版式的格子数量时，未使用的位置会保留白色，这是正常设计。

### 为什么无法访问某个文件？

ImgDeck 使用 macOS App Sandbox，只能读取用户通过系统文件窗口主动选择的图片。请点击“添加图片”重新选择需要的文件。

### 为什么修改分辨率后预览没有立即变化？

修改分辨率后，请再次点击“预览”，新设置才会应用到生成结果。

### 图片会上传到服务器吗？

不会。图片排版、预览和保存均在当前 Mac 本地完成。
