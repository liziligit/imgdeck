# ImgDeck App Review 审核说明

## 审核备注（可直接粘贴）

ImgDeck 是一款完全在设备本地运行的 A4 图片排版工具，不需要登录，不包含广告、订阅或 App 内购买，也不收集或上传用户数据。

核心功能测试步骤：

1. 启动 App，点击“添加图片”。
2. 在 macOS 系统文件窗口中选择 1～9 张常见格式的图片。
3. 使用“上移”“下移”调整图片顺序。
4. 选择一种“行 × 列”A4 版式。
5. 设置分辨率，点击“预览”。
6. 选择 PNG 或 JPG，点击“保存图片”。
7. 在系统保存窗口中选择保存位置。

App 已启用 App Sandbox，只使用 User Selected File 的 Read/Write 权限。它只能读取用户通过系统文件窗口明确选择的图片，并只能向用户明确选择的保存位置写入结果。

## 登录信息

- 是否需要登录：否。
- 演示账号：不适用。

## 网络与数据

- App 不需要网络连接。
- App 不上传图片。
- App 不收集个人数据、使用数据或诊断数据。
- App 不包含第三方广告、分析或跟踪 SDK。
