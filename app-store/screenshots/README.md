# Mac App Store 截图清单

此目录只应放置最新 Swift 版 ImgDeck 的商店截图，不要使用旧 Python/Tkinter 版本截图。

## 规格

- 格式：PNG 或 JPG；
- 数量：1～10 张，建议 3～5 张；
- 比例：16:10；
- 所有截图建议统一使用同一尺寸。

可选尺寸：

- 1280×800；
- 1440×900；
- 2560×1600；
- 2880×1800。

## 建议文件名与内容

1. `01-main-window.png`：软件主界面，展示清晰的完整窗口。
2. `02-images-and-layout.png`：导入多张示例图片并选择版式。
3. `03-a4-preview.png`：展示生成后的 A4 预览。
4. `04-layout-options.png`：展示不同版式选择能力。
5. `05-export.png`：展示 PNG/JPG 格式与保存操作。

## 拍摄检查

- 使用正式准备提交的 Swift 版本；
- 不出现测试路径、个人姓名、通知或其他隐私信息；
- 不使用无权公开的图片素材；
- 界面文字清晰，不对截图进行误导性修改；
- 截图中的功能必须与提交版本一致；
- 截图前检查应用图标已更新且没有白色边界。
