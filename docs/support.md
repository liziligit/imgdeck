## 系统要求

- macOS 13.0 或更高版本；
- 当前发布目标以 App Store 页面显示的兼容机型为准。

## 使用方法

1. 点击“添加图片”，选择 1～9 张图片。
2. 调整图片顺序并选择 A4 版式。
3. 设置分辨率与 DPI/DPCM 单位。
4. 点击“预览”检查效果。
5. 选择 PNG 或 JPG，点击“保存图片”。

## 常见问题

### 为什么有些位置是白色？

当图片数量少于版式格子数量时，未使用的位置会保留白色。

### 图片会上传到服务器吗？

不会。图片排版、预览和保存均在当前 Mac 本地完成。

### 为什么修改分辨率后预览没有立即变化？

请再次点击“预览”，新设置才会应用到生成结果。
